# Kong - Ambiente local de desarrollo

Instrucciones de instalación:

```bash
git clone http://148.227.152.27/cesar.guillen/kong.git

cd kong

# descarga de imágenes y ejecución de migraciones
docker compose up

# ejecución
docker compose up -d
```

# Configuración

```bash
#Iniciar Konga
http://localhost:1337/

```
